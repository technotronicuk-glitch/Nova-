Driver Integration

Nova Horizon OS utilizes an auto-detect kernel. Use our C++ library to ensure your module is recognized instantly.

#include <nova_module.h>

void onModuleConnect() {
    Nova::InitializeBus(G1_RAIL);
    Nova::RequestPower(3.3v);
}
